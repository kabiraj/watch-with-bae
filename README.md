# 🎥 WatchWithBae

A Chrome extension and web app to help long-distance couples watch movies together, inspired by 24/7 webapp but with a cleaner UI and only the features we need.

## 🚀 Project Status

This project is in the early stages of development.  
Follow along as I build it step by step and document my learning journey!

## ✨ Goals

- Detect videos on streaming sites with a Chrome extension
- Send video links to a custom web app
- Synchronized video playback and chat
- Clean, modern, and easy-to-use interface

## 🛠️ Tech Stack

- Chrome Extension (JavaScript)
- React (Frontend)
- Node.js + Socket.io (Backend, planned)
- MongoDB or Firebase (Database, planned)

## 📚 Learning Focus

- Building browser extensions from scratch
- Real-time web communication (WebSockets)
- Clean code and documentation practices
